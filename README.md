# Student Gradebook Analyzer

## Project Overview

This project demonstrates how to analyze student grades using Pandas and NumPy.

The project stores grades in a CSV file and calculates various statistics for students and assignments.

---

## Features

- Load student grades from CSV
- Calculate average score
- Calculate median score
- Calculate maximum score
- Calculate minimum score
- Sort students by average marks
- Identify top performer
- Calculate assignment-wise statistics
- Use NumPy for numerical calculations

---

## Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## Files

student_grades.csv

Contains student names and assignment marks.

Student_Gradebook_Analyzer.ipynb

Contains the complete analysis.

README.md

Project documentation.

## How to Run

1. Install required libraries

pip install pandas numpy

2. Place student_grades.csv in the same folder as the notebook.

3. Open Jupyter Notebook.

4. Run all cells.
